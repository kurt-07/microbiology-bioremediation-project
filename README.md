# microbiology-bioremediation-project
🌱 Bioremediation of Environmental Pollutants Using Microbial Consortia

📌 Overview

This project demonstrates how microorganisms can be used to degrade environmental pollutants such as oil and organic waste. It highlights sustainable and eco-friendly approaches to environmental cleanup.

🎯 Objective

To evaluate the efficiency of microbial consortia in biodegradation processes and analyze their effectiveness in reducing pollutants.

🧪 Methodology

- Collected soil and water samples from polluted environments
- Isolated microorganisms using nutrient agar
- Conducted biodegradation experiments using oil-contaminated samples
- Observed changes over time (5–10 days)

📊 Results

- Significant reduction in pollutants observed
- Mixed microbial cultures showed faster degradation
- Demonstrated eco-friendly alternative to chemical treatments

🔬 Skills Gained

- Microbial culturing techniques
- Environmental sampling
- Laboratory experimentation
- Data analysis and interpretation

🌍 Applications

- Wastewater treatment
- Oil spill cleanup
- Soil restoration
